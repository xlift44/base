File - Preferences - Settings (CTRL + Запятая)

- расширение закладок: "workbench.editor.wrapTabs" true

- автоопределение тазмера и типа табов: "editor.detectIndentation" false
- количество пробелов в табе: "editor.tabSize" 4
- пробелы вместо табов: "editor.insertSpaces" true

сравнение текста с буфером обмена: CTRL + K, C
